Installation

설치 방법

0

백업

기존 ~/.claude/ 파일을 _backup/에 보관

1

전체 배포

robocopy로 .claude/ 전체 복사

2

설정 병합

settings.json, .mcp.json, CLAUDE.md 보존형 병합

3

배포 검증

install-verify.js로 서브디렉토리 확인

보존형 병합 - 기존 사용자 설정 절대 덮어쓰지 않음

마커 기반 - 배포 구간만 추가/갱신, 개인 설정 보존

자동 백업 - 덮어쓰기 전 직전 1세대 안전 백업

전역 배포 - 한 번 설치로 모든 프로젝트에서 사용

git clone && cd nepes-ai-agents && install.bat
