# 본문 양식: 일반 업무 (기타, 분석, 교육, 문의, 데이터 요청/변경/추가, 권한 요청, 회의 등)

## 적용 요청구분

`기타`, `분석`, `교육`, `문의`, `데이터 요청`, `데이터 변경`, `데이터 추가`, `권한 요청`, `결재 양식 추가`, `회의`, `시스템 환경 설정 및 구축`

## 본문 구조

자유 형식. 작업 내용을 구체적으로 설명한다.

```
{작업 내용을 자유롭게 기술}

- 항목1
- 항목2
- ...
```

## 예시

```
YTAP_MANAGER 프로젝트의 CMMI Level2/3 소프트웨어 형상관리(SCM) 문서를 Obsidian vault에 신규 작성하였다.

작성 항목 (12개)
1. 브랜치 전략 수립 — master + feature 2단계 Trunk-Based 구조
2. 버전 관리 체계 정립 — vMAJOR.MINOR.PATCH 3-Level 체계
3. 커밋 컨벤션 표준화 — type: 설명 형식
...

감사 실행 결과
- CM 형상감사: 6/7 통과
- PPQA 셀프감사: 4/4 통과
```
