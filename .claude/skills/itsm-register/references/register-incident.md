# 본문 양식: 장애/대응 (오류 수정, 현업 대응, 시스템 장애, 배치 오류)

## 적용 요청구분

`오류 수정`, `현업 대응`, `시스템 장애`, `배치 오류`

## 본문 구조

```
[현상]
- 누가 어떤 경로로 연락했는지 (예: Teams, 전화, 직접 확인 등)
- 어떤 장비/시스템에 무슨 문제가 발생했는지

[대응 내용]
- 수행할 조치를 단계별로 기술
```

## 예시

```
[현상]
- 라웨 김재경 스타 Teams 연락
- F-BG01, F-BG02 장비 에러/Down 상황 발생

[대응 내용]
- 장비 상태 관련 Remote 가능 여부 확인
- 장비 Down 원인 추정 논의
- 단계별 조치(1차/2차 대응) 협의
- 장비 안정화 및 후속 대응 논의
```
