알고리즘

SPC Engine

4가지 통계적 공정 관리 알고리즘으로 드리프트, 급변, 패턴, 변점을 다각도로 감지.

EWMA

점진적 드리프트

지수 가중 이동 평균. 소규모 공정 이탈을 조기 감지. λ로 민감도 조절.

z_t = λ·x_t + (1-λ)·z_{t-1} UCL = μ + L·σ·√(λ/(2-λ))

CUSUM

작은 이탈 누적

누적 합산. 작은 공정 이탈도 시간 누적으로 확실히 감지. k(슬랙), h(한계) 파라미터.

C⁺ = max(0, C⁺_{t-1} + (x_t-μ)/σ - k) 알람: C⁺ > h 또는 C⁻ > h

WECO

패턴 감지

Western Electric 5가지 규칙. 단순 경계 초과 외 연속성·경향성 패턴을 포착.

R1: 3σ 초과 R2: 9연속 동방향 R4: 2/3이 2σ 초과 R5: 4/5이 1σ 초과

PELT

변점 감지

Pruned Exact Linear Time. 베이스라인 드리프트 자동 감지 후 재설정 트리거. DBSCAN 클러스터 가드.

변점 감지 → 베이스라인 무효화 → 클러스터 시더 재실행
