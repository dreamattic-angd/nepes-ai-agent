SPC 엔진

SPC Algorithms

EWMA

점진적 드리프트

지수 가중 이동 평균. λ로 민감도 조절. 소규모 이탈 조기 감지.

z_t = λ·x_t + (1-λ)·z_{t-1} UCL = μ + L·σ·√(λ/(2-λ))

CUSUM

누적 이탈

누적 합산. 작은 이탈도 시간 축적으로 확실히 감지. k·h 파라미터.

C⁺ = max(0, C⁺_{t-1} + (x_t-μ)/σ - k) 알람: C⁺ > h 또는 C⁻ > h

WECO

패턴

Western Electric 5규칙. 연속성·경향성 패턴 포착.

R1: 3σ 초과 R2: 9연속 동방향 R4: 2/3이 2σ초과 R5: 4/5이 1σ초과

PELT

변점 감지

Pruned Exact Linear Time. 베이스라인 드리프트 자동 감지 → 재설정.

변점 감지 → 베이스라인 무효화 → DBSCAN 클러스터 시더 재실행
